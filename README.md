# 📄 PDF Assistant

## 📸 Screenshots

<img width="1128" height="782" alt="Screenshot 2026-07-22 232917" src="https://github.com/user-attachments/assets/f04af0ad-7b8d-4375-837f-4a5c4d8ee191" />


An AI-powered PDF chatbot that lets users upload documents and ask natural language questions. The application uses a **Retrieval-Augmented Generation (RAG)** pipeline with semantic search and vector embeddings to provide accurate, context-aware answers from uploaded PDFs.

---

## 🚀 Features

- 📤 Upload and process PDF documents
- 🤖 AI-powered conversational Q&A
- 🔍 Semantic search using vector embeddings
- 🧠 Retrieval-Augmented Generation (RAG)
- 🔐 Secure authentication with Supabase
- ⚡ Fast AI responses with Google Gemini
- 💬 Context-aware conversations
- ☁️ Cloud-based vector storage using Pinecone
